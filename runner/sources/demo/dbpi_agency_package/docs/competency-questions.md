# DBPI Competency Questions

These competency questions define the kinds of questions that the City of Red Vale Department of Building Permits and Inspections (DBPI) ontology is expected to answer. They are written as business-facing questions, not as technical query statements.

## 1. Organization and responsibility

1. Which division and work unit currently own a given permit application?
2. Which employee is currently assigned to a permit application, plan review, inspection, or violation case?
3. Which employees belong to a given work unit?
4. Which employees serve in the role of Inspector, Plan Reviewer, Permit Technician, Code Enforcement Officer, Records Officer, or Hearing Officer?
5. Which employee manages a given unit?
6. Which units report to a given division or department?
7. Which matters are currently assigned to a given employee across permitting, inspections, and enforcement?
8. What are the contact points, telephone numbers, and URLs for a given department or division?

## 2. Jurisdiction, property, and parties

9. Which property, parcel, and jurisdiction are associated with a permit application or violation case?
10. Who owns a given property?
11. Which applications, permits, inspections, and violation cases concern a given property?
12. Which applications or cases involve a given person or organization as applicant, owner, contractor, or other responsible party?
13. Which properties fall within a given jurisdiction?
14. Which addresses refer to the same parcel or property record?

## 3. Public services and intake

15. Which public service is an application requesting?
16. Which applications are pending completeness review?
17. Which applications were submitted by a given organization within a specified period?
18. Which supporting documents were submitted for a given application?
19. Which applications are missing required documents, fees, or other requirements?
20. Which applications remain in incomplete status beyond the published service target?

## 4. Permit application processing

21. Which applications are in submitted, in-review, correction-required, approved, denied, withdrawn, or expired status?
22. Which applications requested a given permit type or work type?
23. Which permit applications have open review comments?
24. Which plan reviews are attached to a given application?
25. Which reviewer made a given review comment?
26. Which code sections were cited in review comments for a given application?
27. Which applications required multiple rounds of corrections before decision?
28. Which applications were denied, and what reasons or cited code sections support that denial?

## 5. Issued permits and authorizations

29. Which issued permits were based on a given application?
30. Which permits are active, expired, suspended, revoked, closed, or otherwise not in force?
31. Which permit authorizes a given permit type or work type at a property?
32. What are the issued date and expiration date of a permit?
33. Which requirements, conditions, documents, or fees are attached to a given permit?
34. Which permits at a given property remain open?
35. Which permits issued to the same applicant or contractor are approaching expiration?

## 6. Fees and payments

36. Which fees were assessed for a given application or permit?
37. Which fees have been paid, and on what dates?
38. Which applications or permits have outstanding unpaid fees?
39. What is the total amount assessed and paid for a permit lifecycle?
40. Which applicants or organizations account for the largest unpaid fee balances?

## 7. Inspection scheduling and results

41. Which inspection requests exist for a given permit?
42. Which inspections are scheduled for a given date, property, or inspector?
43. Which inspections fulfill which inspection requests?
44. Which inspection type was requested or performed?
45. Which inspections resulted in pass, partial pass, correction required, cancelled, not ready, or failed status?
46. Which inspector performed a given inspection?
47. Which inspections cited specific code sections?
48. Which permits have repeated failed inspections?
49. Which properties have repeated not-ready or cancelled inspection outcomes?

## 8. Enforcement and compliance

50. Which failed inspections resulted in or are related to violation cases?
51. Which violation cases are open, closed, appealed, or pending hearing?
52. Which notices of violation have been issued for a given property, permit, or case?
53. Which corrective actions are attached to a violation case?
54. Which corrective actions remain overdue or incomplete?
55. Which properties have recurring violations across multiple permits or time periods?
56. Which code sections are cited most often in violation matters?
57. Which cases were opened from complaints rather than from routine inspections?
58. Which cases escalated from warning or correction notice to formal enforcement?

## 9. Appeals, hearings, and adjudication

59. Which appeals are attached to a given violation case?
60. Which appeals are pending, sustained, denied, withdrawn, or remanded?
61. Which hearing officer is assigned to a given hearing?
62. On what date is a hearing scheduled?
63. Which notices, documents, inspections, and cited code sections support the case being heard?
64. Which violation cases resulted in appeal?
65. Which appeal outcomes are associated with particular code sections, units, or decision types?

## 10. Documents, records, and traceability

66. Which documents support a given application, permit, inspection, violation case, or appeal?
67. Which document numbers and document dates are associated with a matter?
68. Which decisions can be traced to a responsible employee, a unit, and one or more cited code sections?
69. Which records exist for the complete lifecycle of work at a given property?
70. Which permits, inspections, notices, and certificates form the history of a parcel or address?
71. Which external references link a DBPI record to another system or agency?

## 11. Performance, workload, and management

72. How many active matters are assigned to each work unit?
73. Which cases have been open the longest in each stage of the lifecycle?
74. Which reviewers or inspectors have the highest workload volume?
75. Which permit types generate the most review comments or failed inspections?
76. Which properties, applicants, or contractors are associated with the highest volume of repeat activity?
77. Which case categories most often lead to appeal or escalation?

## 12. Policy and compliance assurance

78. Are comparable cases being treated consistently across units and employees?
79. Can each material decision be connected to the controlling record, cited code section, and responsible actor?
80. Are enforcement actions being taken proportionally to the severity and recurrence of noncompliance?
81. Are permits being closed only after all required inspections, fees, and conditions are satisfied?

## Coverage note

Together, these competency questions exercise the ontology modules for:

- organizational structure and staff responsibility;
- properties, parcels, parties, and jurisdictions;
- application intake and plan review;
- permit issuance and lifecycle status;
- fee and payment tracking;
- inspections and inspection results;
- enforcement, corrective action, appeals, and hearings; and
- records traceability and management reporting.
